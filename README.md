# Beauty-Parlor-Management-System
Beauty Parlor Management System built with HTML5, CSS3, Bootstrap, JavaScript, jQuery, PHP, and MySQL (via SQL Server Management Studio). Features include appointment scheduling, customer enquiries, admin dashboard, authentication, and service listings. Live Demo: bpms.free.nf
## 🚀 Features
- Appointment scheduling
- Customer enquiries
- Admin dashboard
- Authentication & user management
- Service listings

## 🛠️ Tech Stack
- HTML5, CSS3, Bootstrap
- JavaScript, jQuery
- PHP, MySQL (SQL Server Management Studio)

## 📦 Installation Guide
1. Clone the repository:
   ```bash
   git clone https://github.com/parveennazreen5801-cloud/Beauty-Parlor-Management-System.git 2. Move files to your local server (`htdocs` for XAMPP).
3. Import the database file (`bpms.sql`) into MySQL.
4. Start Apache & MySQL in XAMPP.
5. Open `http://localhost/Beauty-Parlor-Management-System` in browser.
