# Beauty Parlor Management System (BPMS)

A web application built with **HTML5, CSS3, Bootstrap, JavaScript, jQuery, PHP, and MySQL (via SQL Server Management Studio)** to manage beauty salon operations.  
Features include appointment scheduling, customer enquiries, admin dashboard, authentication system, and service listings.  

**Live Demo:** [bpms.free.nf](http://bpms.free.nf)

## Features
- Appointment booking
- Customer enquiry form
- Admin dashboard with authentication
- Service listings
- Responsive design

## Tech Stack
- HTML5, CSS3, Bootstrap
- JavaScript, jQuery
- PHP, MySQL (SQL Server Management Studio)
